第四单元 运算符与语句 part1 运算符的使用

动手任务1：算术运算符的使用

1.算术运算符：+-*/%
2.加号的使用: 相加，字符串连接，如果有连接的使用，后面都是连接。

        System.out.println(3 + 4);//3+4=7；
        System.out.println("china"+3 +4);//china34，第一个加号是连接，第二个也是连接
        System.out.println(3+4+"china"+3 +4);//7china 第一个加号是相加，第二个是连接。后面的都是相连了

3.减号的使用： 

        System.out.println(1-0.9);//浮点型数转换成二进制的时候会有误差，0.09999999999999998
        System.out.println(2.0-1.9);//同样有误差，每次答案都不相同

3.除号的使用
整数/整数=整数

        System.out.println(3/2);//1
        System.out.println(2/3);//0
        System.out.println(2.0/3);//0.66666666
//        System.out.println(3/0); //整数/0位错误
        System.out.println(3.0/0);//Infinity,无穷大

4.取余数的使用 

        System.out.println(3%2);//1
        System.out.println(2%3);//2
        System.out.println(2%3.0);//2.0
        System.out.println(3.0%0);//NAN Not A Number

5:自加自减得使用： 

        int age =10;
        System.out.println(++age);//11
        int grade =10;
        System.out.println(grade++);//10
        //System.out.println(300+); 常亮不可以进行自加操作

动手任务二：赋值运算符

1.赋值=
不要超范围，不要类型不匹配
2.扩展的赋值运算符：
+=即自加
i+=2;
等同于
i=i+2; 

        short weight = 100;// !的意思是，没有用过该变量！
        weight += 1;// 存在着强制类型转换,不需要分配新的单元；
        System.out.println(weight = weight + 1);// 错误

动手任务三：关系运算符

出来的结果是Boolean类型

大于
= 大于或等于
< 小于
<= 小于或等于
== 是否相等

!= 是否不等 ：关系运算符 

        System.out.println(3!=0);//true
        System.out.println(3 >= 7);//false

动手任务四：逻辑运算符

````java
public class LogicOperato {
public void calc() {
int a = 100;
int b = 200;

  if(a<38 &++b>10) {
  }
  System.out.println(b);//b=201

    /**
     * 短路与&&，前面的式子可以确定结果了，后面的就可以不执行了
     * 短路 与的效率高
     * 一般&，就要执行所有的了；
     */
    if (a < 38 && ++b > 10) {
    }
    System.out.println(b);//b=201

}

```

& 与 && 的区别：& 和 | 左右两边的式子一定会执行（比较笨），&& 和 || 只要左边的式子能得出结果，右边的式子就不会执行（比较聪明）
