动手任务2：转义字符的使用

口述1：成员变量，局部变量是什么，怎么区分？

成员变量是整体的，在整个类都生效的；局部变量：方法和语句块内定义的的变量。
并且局部变量必须初始化（赋值），否则会报错；
分辨的方法就是看他们的出生地点；

口述2：什么是标识符？标识符命名归则是什么？

Java 中标识符是为方法、变量或其他用户定义项所定义的名称。标识符的命名规则：以字母，$,_开头的，后面可以是数字。且不能是关键字。

口述3：什么是关键字？特点？多少个？

Java关键字是电脑语言里事先定义的，有特别意义的标识符，有时又叫保留字，还有特别意义的变量。
特点是，他们都是小写的了。
一共有53个（其中包含两个保留字const，goto），true，false，null也算是。

口述4：Java数据类型分类？分别是什么？


共11种：8种基本类型：
boolean：长度是多少？
大小尚不明确
byte ：一个字节占用8位 。范围：- 128127；byte age =13；
char ：两个字节占用16位。 范围：0 65535 。字符编码：Unicode
一个汉字几个字符？ 一个汉字一个字符！
只能存放一个字符，超过一个字符就会产生编译错误
short： 两个字节，16位数。 范围： -32768~32767.
int：四个字节，32位。 -2(n-1)–2(n)-1 n=32
long：八个字节，64位
后面L加不加啊？
如果超过int的最大值了，必须加！
float :小数，四个字节，32位 。
F或者f可以省去吗？
不可以省去的哦；
double：八个字节，64位数
F，f可以去。
*小数默认类型为double，整数默认类型为int *

引用数据类型3种：
类，接口，数组；

口述5： String是不是内置程序类型，是不是关键字？

String不是内置程序类型，也不是关键字！

口述6：20，9.8的默认类型是什么？

20的默认类型是int ，整数默认是int
9.8的默认类型是double，小数默认是double

口述7： float数据的f或者F可以省去吗？

不可以的

口述8：boolean类型的数据长度是多少？

根据官网所说，长度还没有明确的定义了。

口述9：Java基本数据类型转换规则？

有一种类型不能转换，那就是Boolean。
数据类型转换归则:低级类型转高级类型，自动；高级类型转低级类型，必须强制
在强制转换的过程中，很有可能会发生数据的丢失，具体的情况还需要看情况而定，位数的截取等情况，很有可能会发生。

课后任务：

Teacher类属性：教师年龄（byte）、教师月薪（short）、教师年薪（int）、 教师工号（long）、教师体重（float）、教师身高（double）、教师性别（char）、是否是教授（boolean）
覆盖知识点：标识符、关键字和基本类型；标识符命名规则（面试题）、

给每个属性提供setter/getter方法

public class Teacher {
    String name;
    byte age;
    short monSalary;
    int yeaSalary;
    long teacherId;
    float weight;
    double tall;
    char sex;
    boolean teach;

    /**
     * 设置属性
     * 
     * @param n 名字
     */
    public void setName(String n) {
        name = n;
    }

    /**
     * 取回属性
     * 
     * @return 名字
     */
    public String getName() {
        return name;
    }

    /**
     * 设置年龄属性
     * 
     * @param a 年龄
     */
    public void setAge(byte a) {
        age = a;
    }

    /**
     * 取回年龄属性
     * 
     * @return 年龄
     */
    public byte getAge() {
        return age;
    }

    /**
     * 设置月薪
     * 
     * @param m 月薪
     */
    public void setMon(short m) {
        monSalary = m;
    }

    /**
     * 取回月薪
     * 
     * @return 月薪
     */
    public short getMon() {
        return monSalary;
    }

    public void setYea(int y) {
        yeaSalary = y;
    }

    public int getYea() {
        return yeaSalary;
    }

    public void setId(long Id) {
        teacherId = Id;
    }

    public long getId() {
        return teacherId;
    }

    public void setWeight(float w) {
        weight = w;
    }

    public float getWeight() {
        return weight;
    }

    public void setTall(double t) {
        tall = t;
    }

    public double getTall() {
        return tall;
    }

    public void setSex(char s) {
        sex = s;
    }

    public char getSex() {
        return sex;
    }

    public void setTeach(boolean flag) {
        teach = flag;
    }

    public boolean getTeach() {
        return teach;
    }

    public static void main(String[] args) {
        /**
         * 
         */
        Teacher p1 = new Teacher();
        /**
         * 调用setName,设置属性;
         */
        p1.setName("刘欢");
        p1.setAge((byte) 12);
        p1.setMon((short) 100);
        p1.setYea(1200);
        p1.setId(23102);
        p1.setWeight(120f);
        p1.setTall(180);
        p1.setSex('男');
        p1.setTeach(false);
        /**
         * 调用getName，取回属性;
         */
        System.out.println("姓名  " + p1.getName());
        System.out.println("年龄 " + p1.getAge() + "岁");
        System.out.println("月薪" + p1.getMon() + "$");
        System.out.println("年薪" + p1.getYea() + '$');
        System.out.println("工号"+ p1.getId());
        System.out.println("体重" + p1.getWeight() +"kg");
        System.out.println("身高"+p1.getTall() +"cm");
        System.out.println("性别"+ p1.getSex());
        System.out.println("是否教课"+ p1.getTeach());
    }

}
